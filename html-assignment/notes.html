HTML tags with description

p
this is paragraph tag and it is a paired tag so it must be closed with /p.
sub
this is a tag defines text as being subscript like H2O.
sup
this is a tag defines text as being superscript like 10th.
body
this is the main body of the document, this is what we see in our browsers. All the content of the web page must be added inside the opening body and closing /body.
nav
this is the navigation of the page, this will contain the links to other pages of the website.
